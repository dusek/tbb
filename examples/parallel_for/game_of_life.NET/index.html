Overview

"Game of life" demonstrates interoperability TBB and .NET.
This program runs 2 simultaneous instances of the classic "game of Life”. One of these instances is calculated using sequential programming. The other is calculated using parallel programming via TBB.

Files

Evolution.cpp
File contain example class hierarchy.
Game_of_life.cpp
File contain program entry point and other source that not related to logical structure of example.
Update_state.cpp
File contain 2 type of source code for calculating steps in program. First type use SSE, second is ordinary C++ code.

Directories
